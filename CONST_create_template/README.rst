package_gmf project
===================

Read the `Documentation <https://camptocamp.github.io/c2cgeoportal/2.5/>`_

Checkout
--------

.. code::

   git clone git@github.com:camptocamp/package_gmf.git

   cd package_gmf

Build
-----

.. code::

  ./build

Run
---

.. code::

   docker-compose up -d
